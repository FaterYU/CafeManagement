# CafeManagement

SCNU&amp;UoA Cafe Management

## 快速开始

- **安装Node.js**

  - **[Node.js v16.14.2](http://nodejs.cn/)**
  - **[npm v8.5.5]**

### 前端

#### 安装依赖

```bash
    cd cfm-vue
    npm i
```

#### 快速启动

```bash
    npm run electron:serve
```
